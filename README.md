## Development

Create a `.env` file in the web folder with the secrets:

```ini
REACT_APP_CONTENTFUL_SPACE_ID=
REACT_APP_CONTENTFUL_ACCESS_TOKEN=
REACT_APP_SENTRY_DSN=
REACT_APP_SENTRY_RELEASE=dev
```
